What's Live Right Now

This is a soft launch.
Here's exactly what's working.

We're in active development. These features are running in production today on real San Francisco data.

Downtown SF Business Portal

Live business tracker ingesting SF Business Registrations via DataSF. 2,710 active businesses, month-over-month trend charts, category breakdown, and spatial correlation insights. Running at web-five-omega-87.vercel.app/cbd/downtown.

Property Intelligence Dashboard

Any SF building by Assessor Parcel Number. No login. Live permits (DBI), 311 requests, nearby businesses, SFMTA transit stops, and an AI redevelopment signal. Try it at /property/3713007 (1 Market St).

Transit Scoring

Walk-access scoring using the Vale 2015 / EPA Smart Location Database distance-decay model across an 800m pedestrian shed. Weighted by stop type (BART, Muni Metro, local bus). 1 Market St scores 100/100.

Nextspace 3D (UAT)

CityPulse intelligence packages are parsing correctly in the Nextspace UAT environment. Full city-scale binding is pending an APN export from CyberCity 3D. The 3D map with 72 FiDi footprints is live locally today.

AI Briefings

Claude generates district briefings, business category insights, and redevelopment signals on demand. All API calls are cached per session. Anthropic API costs scale with usage — this is factored into pricing.
